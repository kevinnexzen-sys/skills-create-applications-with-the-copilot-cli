# GitHub Copilot CLI Skills - Feature Roadmap

## Overview
This repository contains learning exercises for creating applications with GitHub Copilot CLI.

## Feature Requests

### 1. CALCULATOR ✅
**Status**: COMPLETED
**Description**: Build a calculator application using Copilot CLI

**Implementation Guide**:
```bash
# Run calculator
npm run calculator

# Features:
- add(a, b)
- subtract(a, b)
- multiply(a, b)
- divide(a, b)
```

**Acceptance Criteria**:
- [x] CLI calculator with basic operations
- [x] Input validation
- [x] Error handling
- [x] Interactive REPL interface

---

### 2. CLI (Command-Line Interface) ✅
**Status**: COMPLETED
**Description**: Develop a CLI tool template and utilities

**Implementation Guide**:
```bash
# Run CLI tool
npm run cli [command] [args]

# Features:
- Command parsing
- Help documentation
- Error handling
- Multiple commands (greet, list, info, calc, help)
```

**Acceptance Criteria**:
- [x] Command parsing
- [x] Help documentation
- [x] Error handling
- [x] Multiple working commands

---

### 3. AGEN (Agent Framework) ✅
**Status**: COMPLETED
**Description**: Create an AI agent framework using Copilot

**Implementation Guide**:
```bash
# Run agent
npm run agent

# Features:
- Agent lifecycle management
- Task queuing and orchestration
- Decision-making logic
- Integration interfaces
```

**Acceptance Criteria**:
- [x] Agent base class
